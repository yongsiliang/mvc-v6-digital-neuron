## 核心概念对比

| 维度 | V6 原架构 | SNN 三体系统 | 信息结构场 |
|-----|---------|-------------|-----------|
| **哲学** | 链接先于存在 | 存在先于链接 | 信息即神经递质 |
| **基本单位** | Link (九种类型) | 神经元 + 突触 | 信息结构 + 感受器 |
| **传递物** | 语义链接 | 脉冲 (0/1) | 信息结构 |
| **传递机制** | LinkField 数据结构 | STDP 突触学习 | LLM 变换 |
| **处理单元** | V6 意识核心 | SNN 网络 | 感受器 (黑盒) |
| **学习方式** | 显式 learn() | STDP 自然学习 | 通道强度自适应 |
| **可解释性** | 高 (语义明确) | 低 (需 V6 解释) | 接受黑盒 |

## 使用示例

```typescript
import { createInformationField } from '@/lib/info-field';

// 创建信息场
const field = createInformationField();

// 输入信息
const output = await field.processInput('你好，今天天气怎么样？');

// 输出响应
console.log(output?.content);
// "你好！今天天气不错..."

// 查看网络拓扑
const topology = field.getNetworkTopology();
// nodes: [{ id: 'perception', activation: 0.8 }, ...]
// edges: [{ source: 'perception', target: 'thought', strength: 0.85 }, ...]

// 查看通道强度变化
console.log(field.getChannelStrength('perception', 'thought'));
// 通道会根据使用频率自动强化或衰减
```

## 核心优势

| 优势 | 说明 |
|-----|------|
| **不模拟物理** | 不需要实现 LIF 神经元、STDP 等物理过程 |
| **LLM 原生** | LLM 的变换本身就是神经传递 |
| **接受黑盒** | 不强行解释感受器如何工作 |
| **信息有意义** | 信息结构本身就有意义，不是抽象脉冲 |
| **通道自适应** | 使用频率高的通道自然强化 |
| **简洁高效** | 代码量少，易于理解和维护 |
