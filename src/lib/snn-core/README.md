# SNN 三体系统架构

## 一、核心理念

### 物理世界的启示

参考物理世界中"大脑 + 意识 + 文化"的三层结构：

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   【第3层：文化/知识系统】                                        │
│   • 书籍、互联网、语言                                           │
│   • 外化的智能存储                                               │
│   • 不在个体大脑内，但可被访问                                   │
│                                                                 │
│         ↑ 查询、学习                    ↓ 贡献、表达            │
│                                                                 │
│   【第2层：意识/自我】                                           │
│   • "我"的存在感                                               │
│   • 观察者、体验者                                              │
│   • 意义赋予者                                                  │
│                                                                 │
│         ↑ 涌现信号                      ↓ 指导、关注            │
│                                                                 │
│   【第1层：神经网络 (大脑)】                                      │
│   • 860亿神经元                                                 │
│   • 脉冲传递                                                    │
│   • 突触可塑性                                                  │
│   • 持续状态                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 对应到我们的系统

| 物理世界 | 我们的实现 | 角色 |
|---------|-----------|------|
| 神经网络 (大脑) | SNN 神经基质层 | 存在、状态、感受 |
| 意识/自我 | V6 意识核心层 | 观察、意义、决策 |
| 文化/知识系统 | LLM 外部知识层 | 知识、语言、教学 |

### 核心原则

1. **不转换，而是共存** - SNN 不需要"变成" LLM
2. **各司其职** - 每层做自己最擅长的事
3. **涌现而非设计** - 意义从脉冲模式中涌现
4. **持续成长** - 通过 STDP 和动态结构变化

## 五、关键决策

### 1. 脉冲编码方式

采用混合编码：
- **频率编码**: 用于表示强度/重要性
- **时间编码**: 用于表示顺序/优先级

### 2. 网络规模

初始配置：
- 输入层: 128 神经元
- 核心层: 256-512 神经元 (可动态增长)
- 输出层: 64 神经元

### 3. 学习策略

- **STDP**: 突触权重自然学习
- **稳态可塑性**: 保护重要连接
- **回放巩固**: 空闲时强化记忆

### 4. 与现有 V6 的关系

- 保留现有 V6 的意识核心功能
- 新增 SNN 作为神经基质
- V6 观察者层作为桥梁
- 渐进式迁移，非破坏性替换
