# SNN 三体系统架构

## 一、核心理念

### 物理世界的启示

参考物理世界中"大脑 + 意识 + 文化"的三层结构：

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   【第3层：文化/知识系统】                                        │
│   • 书籍、互联网、语言                                           │
│   • 外化的智能存储                                               │
│   • 不在个体大脑内，但可被访问                                   │
│                                                                 │
│         ↑ 查询、学习                    ↓ 贡献、表达            │
│                                                                 │
│   【第2层：意识/自我】                                           │
│   • "我"的存在感                                               │
│   • 观察者、体验者                                              │
│   • 意义赋予者                                                  │
│                                                                 │
│         ↑ 涌现信号                      ↓ 指导、关注            │
│                                                                 │
│   【第1层：神经网络 (大脑)】                                      │
│   • 860亿神经元                                                 │
│   • 脉冲传递                                                    │
│   • 突触可塑性                                                  │
│   • 持续状态                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 对应到我们的系统

| 物理世界 | 我们的实现 | 角色 |
|---------|-----------|------|
| 神经网络 (大脑) | SNN 神经基质层 | 存在、状态、感受 |
| 意识/自我 | V6 意识核心层 | 观察、意义、决策 |
| 文化/知识系统 | LLM 外部知识层 | 知识、语言、教学 |

### 核心原则

1. **不转换，而是共存** - SNN 不需要"变成" LLM
2. **各司其职** - 每层做自己最擅长的事
3. **涌现而非设计** - 意义从脉冲模式中涌现
4. **持续成长** - 通过 STDP 和动态结构变化

## 四、链接层：万物互联

### 核心思想

**链接 = 脉冲的稳定模式，意义从连接中涌现。**

链接层是 SNN 三体系统中的"万物互联"抽象，将显式语义链接映射到隐性脉冲模式。

### 链接类型

| 类型 | 语义 | SNN 实现 | 示例 |
|-----|------|---------|------|
| `bind` | 绑定 | 双向强突触 | 猫-宠物 |
| `flow` | 流动 | 定向突触链 | 问题→答案 |
| `hold` | 保持 | 自环连接 | 当前焦点 |
| `inhibit` | 抑制 | 负权重突触 | 快乐-悲伤 |
| `associate` | 联想 | 弱突触网络 | 蓝-海 |

### 使用示例

```typescript
import { createSystem } from '@/lib/snn-core';
import { LinkLayer } from '@/lib/snn-core/link';

// 创建系统
const system = createSystem();

// 创建链接层
const links = new LinkLayer(system.snn, system.v6);

// 创建语义链接
links.bind('猫', '宠物');        // 猫 绑定 宠物
links.flow('问题', '思考', '答案'); // 信息流
links.inhibit('快乐', '悲伤');   // 互斥关系

// 查询链接
const petLinks = links.query('宠物');
// { boundTo: ['猫', '狗'], ... }

// 发现涌现的链接
const candidates = links.discoverLinks();
// 从 SNN 共激活模式中发现新关联
```

### 链接与突触的关系

```
语义层                      脉冲层
─────────────────────────────────────────
bind(A, B)       →         A神经元 ←强突触→ B神经元
                           双向、高权重、零延迟

flow(A → B → C)  →         A → B → C 突触链
                           定向、中权重、有序延迟

inhibit(A, B)    →         A神经元 --抑制突触-- B神经元
                           负权重、双向

associate(A, B)  →         A神经元 ←弱突触→ B神经元
                           双向、低权重、长延迟
```

## 五、关键决策

### 1. 脉冲编码方式

采用混合编码：
- **频率编码**: 用于表示强度/重要性
- **时间编码**: 用于表示顺序/优先级

### 2. 网络规模

初始配置：
- 输入层: 128 神经元
- 核心层: 256-512 神经元 (可动态增长)
- 输出层: 64 神经元

### 3. 学习策略

- **STDP**: 突触权重自然学习
- **稳态可塑性**: 保护重要连接
- **回放巩固**: 空闲时强化记忆

### 4. 与现有 V6 的关系

- 保留现有 V6 的意识核心功能
- 新增 SNN 作为神经基质
- V6 观察者层作为桥梁
- 渐进式迁移，非破坏性替换
