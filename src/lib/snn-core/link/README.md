# 链接层 (Link Layer)

## 核心思想

**链接是脉冲的稳定模式，意义从连接中涌现。**

在 SNN 三体系统中，链接不是一个独立的机制，而是 **突触连接 + STDP 学习 + V6 观察** 的统一抽象。

```
┌─────────────────────────────────────────────────────────────┐
│                  链接的三层实现                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  语义层 (用户/LLM 可见)                                      │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  bind: A 绑定 B  →  持久关联                         │    │
│  │  flow: A 流向 B  →  信息传递                         │    │
│  │  hold: A 保持 B  →  状态保持                         │    │
│  │  inhibit: A 抑制 B  →  互斥关系                      │    │
│  └─────────────────────────────────────────────────────┘    │
│                          ↓                                   │
│  ─────────────────────────────────────────────────────────  │
│                                                              │
│  脉冲层 (SNN 实现)                                           │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  bind   → 强突触 + 双向连接 + 高频共激活             │    │
│  │  flow   → 定向突触链 + 序列放电                      │    │
│  │  hold   → 自环连接 + 持续发放                         │    │
│  │  inhibit → 负权重突触 + 抑制性神经元                 │    │
│  └─────────────────────────────────────────────────────┘    │
│                          ↓                                   │
│  ─────────────────────────────────────────────────────────  │
│                                                              │
│  物理层 (突触/神经元)                                        │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  突触权重变化 (STDP)                                 │    │
│  │  神经元连接生长/修剪                                 │    │
│  │  脉冲时序模式                                        │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 链接类型与 SNN 对应

| 链接类型 | 语义含义 | SNN 实现 | 突触特征 |
|---------|---------|---------|---------|
| `bind` | 绑定/持久关联 | 双向强突触 | 高权重(>1.5)，双向对称 |
| `flow` | 流动/传递 | 定向突触链 | 中权重(0.5-1.0)，有序延迟 |
| `hold` | 保持/记忆 | 自环+Hebbian | 自连接，持续激活 |
| `inhibit` | 抑制/互斥 | 抑制性突触 | 负权重，快速响应 |
| `associate` | 自由联想 | 弱突触网络 | 低权重(0.1-0.3)，广撒网 |

## 使用示例

```typescript
import { LinkLayer } from '@/lib/snn-core/link';

// 创建链接层
const links = new LinkLayer(snn, v6);

// 创建语义链接（会被编码为突触）
links.bind('猫', '宠物');      // 猫 绑定 宠物
links.flow('问题', '思考', '答案');  // 问题 → 思考 → 答案
links.hold('当前焦点');        // 保持当前焦点
links.inhibit('快乐', '悲伤');  // 快乐抑制悲伤

// 查询链接
const petLinks = links.query('宠物');
// 返回: { bound: ['猫', '狗'], flows: [...], inhibits: [] }

// 让链接从 SNN 中涌现
links.learnFromPatterns();  // 分析 SNN 稳定模式，提取新链接
```

## 核心原理

### 1. 链接即突触模式

```typescript
// bind(A, B) 在 SNN 中的实现
A_neurons ←→ B_neurons  // 双向连接
weight(A→B) = high
weight(B→A) = high
delay ≈ 0               // 近乎同时激活

// 当 A 发放时，B 几乎同时发放
// V6 观察到这个模式，理解为"A 和 B 绑定"
```

### 2. 链接强度 = 突触权重

```typescript
interface Link {
  type: 'bind' | 'flow' | 'hold' | 'inhibit' | 'associate';
  source: ConceptId;
  target: ConceptId;
  
  // 映射到 SNN
  synapses: SynapseId[];
  strength: number;  // = avg(synapse.weight)
  
  // 动态调整
  reinforce(): void;   // 通过 STDP 强化
  weaken(): void;      // 通过降低活动弱化
  dissolve(): void;    // 突触修剪
}
```

### 3. 链接涌现

V6 观察器检测到稳定的脉冲模式后，可以"发现"新的链接：

```typescript
// V6 观察
const patterns = v6.detectStablePatterns();

// 例如发现：
// pattern_1: { neurons: ['n1','n2','n3'], coherence: 0.9 }
// pattern_2: { neurons: ['n3','n4','n5'], coherence: 0.85 }
// → 共享神经元 n3 → 可能存在关联

// 反向推断为语义链接
links.discoverFromPatterns(patterns);
// → "概念A" associate "概念B"
```

## 与原架构的关系

| 原架构 | 新架构 | 变化 |
|-------|-------|------|
| 显式 `Link` 类型 | 隐性突触模式 | 从定义变为涌现 |
| `bind/flow/hold` 方法 | 突触权重配置 | 语义→物理映射 |
| 链接存储 | 突触权重 + V6 模式记忆 | 分布式存储 |
| 链接查询 | V6 模式匹配 + 解码 | 意义推断 |

## 优势

1. **动态自组织**: 链接不再静态定义，而是从使用中涌现
2. **泛化能力**: 相似模式自动形成关联
3. **衰减机制**: 不用的链接自然弱化消失
4. **多模态**: 链接可以跨感官模态（视觉↔语言）
