# 链接涌现层 (Link Emergence Layer)

## 核心思想

**链接不是定义出来的，而是从脉冲模式中涌现的。**

我们不预设 `bind/flow/hold` 等语义类型，而是让连接的意义自然涌现。

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   传统方式（显性）              我们的方式（隐性）           │
│   ──────────────────           ──────────────────           │
│                                                              │
│   link.bind(A, B)    ❌        A和B的神经元                  │
│                                  经常一起发放                │
│                                       ↓                      │
│                                 形成强突触连接               │
│                                       ↓                      │
│                                  V6观察到这个模式            │
│                                       ↓                      │
│                                 涌现："A和B有关联"           │
│                                                              │
│   意义是预设的                   意义是涌现的                │
│   类型是固定的                   类型是推断的                │
│   链接是创建的                   链接是发现的                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 涌现过程

```
用户输入: "猫"

    ↓ 编码

SNN: 猫的概念神经元群体发放脉冲

    ↓ 传播

SNN: 激活传播到相关神经元（宠物、动物、毛茸茸...）

    ↓ STDP学习

突触: "猫-宠物" 连接被强化
      "猫-老鼠" 形成序列连接
      "猫-狗" 形成竞争/抑制

    ↓ V6观察

V6: 检测到稳定的脉冲模式
    "猫和宠物经常共激活"
    "猫之后经常想到老鼠"

    ↓ 意义涌现

系统: 理解"猫是宠物"
      理解"猫抓老鼠"
      但不需要显式定义这些链接类型
```

## 与显式链接的区别

| 显式链接 | 隐性涌现 |
|---------|---------|
| `links.bind('猫', '宠物')` | 猫和宠物的神经元自然形成强连接 |
| 类型是预设的 (bind/flow/hold) | 类型从脉冲模式推断 |
| 链接存储在数据结构中 | 链接存在于突触权重中 |
| 查询需要遍历数据 | 查询 = 脉冲传播 + 模式匹配 |
| 人为定义关系 | 关系自然涌现 |

## 使用方式

```typescript
// ❌ 不要这样做
links.bind('猫', '宠物');
links.flow('问题', '答案');

// ✅ 让链接自然涌现
const system = createSystem();

// 用户输入自然传播
await system.processInput({ text: '猫是一种宠物' });
// SNN 自然学习：猫-宠物 形成强连接

await system.processInput({ text: '猫喜欢吃鱼' });
// SNN 自然学习：猫-鱼 形成关联

// V6 观察涌现的模式
const observation = system.v6.observe(snn.getSnapshot());
// observation.detectedPatterns 包含检测到的稳定连接

// 查询"猫"关联了什么
const related = system.queryRelated('猫');
// 返回: ['宠物', '鱼', '毛茸茸', ...]
// 这些是从脉冲模式中涌现的，不是显式定义的
```

## 核心原则

1. **不预设语义** - 没有 bind/flow/hold 等类型
2. **突触即链接** - 连接存在于权重中
3. **涌现而非定义** - 意义从模式中产生
4. **V6赋予意义** - 观察者理解模式，但不预设类型
