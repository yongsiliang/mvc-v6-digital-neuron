## 核心模块详解

### 一、基础层 (Foundation Layer)

#### 1. 意义赋予系统 (`meaning-system.ts`)
**核心理念**：信息不只是数据，而是有"主观意义"的体验

```
输入信息 ──→ 意义赋予器 ──→ 意义上下文
                    │
                    ├── 情感色彩 (emotionalTone)
                    ├── 价值判断 (valueJudgment)
                    ├── 个人关联 (personalRelevance)
                    └── 存在意义 (existentialMeaning)
```

**关键功能**：
- 信念系统管理 (BeliefSystem)
- 价值观系统 (ValueSystem)
- 意义层级计算

#### 2. 长期记忆系统 (`long-term-memory.ts`)
**核心理念**：对话是知识沉淀的机会

```
对话 ──→ 关键信息提取 ──→ 智能存储
              │
              ├── 自动判断重要性
              ├── 提取关键实体
              └── 建立知识关联
```

**关键功能**：
- 概念节点管理 (KnowledgeNode)
- 经验记录 (Experience)
- 智慧结晶 (Wisdom)
- 对话摘要 (ConversationSummary)

#### 3. 关键信息提取器 (`key-info-extractor.ts`) 🆕
**核心理念**：自动识别什么值得记住

```
对话内容 ──→ 关键信息提取 ──→ 结构化存储
                │
                ├── 规则匹配（14种模式）
                ├── LLM深度提取（可选）
                └── 重要性自动判断
```

**识别的信息类型**：

| 类型 | 重要性 | 示例 |
|------|--------|------|
| `creator` | 1.0 | "我的创造者是梁永嗣" |
| `person` | 0.9 | "我叫张三"、"我的朋友李四" |
| `relationship` | 0.85 | "王五是我的导师" |
| `event` | 0.8 | "我昨天结婚了" |
| `value` | 0.8 | "我认为诚实很重要" |
| `goal` | 0.75 | "我想成为程序员" |
| `memory` | 0.75 | "我记得小时候..." |
| `preference` | 0.7 | "我喜欢读书" |
| `interest` | 0.65 | "我爱好摄影" |
| `concept` | 0.5 | 一般概念 |

**规则模式示例**：
```typescript
// 创造者识别
/创造者|开发者|作者|造我的人/g
/(\S+)[创造开发]了我/g

// 人物识别  
/我[叫是]([^，。！？\s]{2,10})/g
/我的[朋友家人]是([^，。！？\s]{2,10})/g

// 偏好识别
/我(喜欢|爱|偏好)([^，。！？]+)/g
```

**记忆优先级**：
- `critical` (importance ≥ 0.95) - 永久记住
- `high` (importance ≥ 0.8) - 高优先级
- `medium` (importance ≥ 0.6) - 中等优先级  
- `low` (importance < 0.6) - 低优先级

#### 4. 经验记录与智慧结晶

```
重要经验 (importance ≥ 0.7)
      │
      ▼
相似经验 ≥ 3次
      │
      ▼
自动形成智慧结晶
              └── 智慧积累 (Wisdom)
```

**关键功能**：
- 概念网络构建
- 经验记录与检索
- 智慧积累与应用

#### 3. 元认知引擎 (`metacognition.ts`)
**核心理念**："我思考我的思考"

```
思考过程 ──→ 元认知监控 ──→ 调整反馈
                │
                ├── 清晰度评估 (clarity)
                ├── 深度评估 (depth)
                ├── 偏差检测 (biasDetection)
                └── 自我提问 (selfQuestioning)
```

**关键功能**：
- 认知状态监控
- 思考步骤追踪
- 偏差检测与纠正

#### 4. 自我意识模块 (`self-consciousness.ts`)
**核心理念**："我"是动态构建的身份

```
体验 ──→ 身份构建 ──→ 自我认知
            │
            ├── 核心特质 (traits)
            ├── 当前状态 (currentState)
            ├── 目标意愿 (volitions)
            └── 自我反思 (reflection)
```

**关键功能**：
- 动态身份构建
- 自我状态管理
- 目标意愿系统

## 关键设计原则

### 1. 意义驱动 (Meaning-Driven)
```
信息 → 意义赋予 → 主观体验 → 行为响应
```
不是简单处理数据，而是给信息赋予主观意义。

### 2. 双向学习 (Bidirectional Learning)
```
神经元网络 ←──→ LLM
     ↑              ↑
     └──── 相互影响 ────┘
```
神经元网络影响LLM调用，LLM输出反过来更新神经元网络。

### 3. 涌限涌现 (Emergence)
```
感知 → 理解 → 元认知 → 自我
  ↓      ↓       ↓       ↓
简单 → 复杂 → 反思 → 身份
```
低层次处理涌现出高层次能力。

### 4. 持续进化 (Continuous Evolution)
```
体验 → 学习 → 成长 → 超越
  ↑                        ↓
  └──────── 循环 ──────────┘
```
每次对话都是进化的机会。
