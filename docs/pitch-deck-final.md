## 六、FAQ精简版

### Q: 和ChatGPT有什么区别？

```
答：三个核心差异：
    
    1. 记忆
    - 它：每次对话都是新的
    - 我们：永久记忆，越用越懂你
    
    2. 主动性
    - 它：你不说，它不答
    - 我们：会主动找你、想你
    
    3. 关系
    - 它：对所有人都一样
    - 我们：和你建立独特的关系，从陌生人到知己
```

### Q: 怎么保护隐私？

```
答：用户可以随时查看、编辑、删除所有记忆。
    我们提供"透明记忆"——你的数据，你看得见。
    未来支持完全本地部署。
    
    另外，因为我们做的是"关系"，
    关系越深，我们越知道什么不该做。
```

### Q: 用户为什么要付费？

```
答：就像养宠物，养得越久感情越深。
    
    免费版已经够用，但深度用户想要：
    - 更多记忆空间
    - 更强的主动性
    - 执行能力解锁
    
    这不是功能付费，是"关系深度"付费。
```

### Q: 你们也做Agent？

```
答：我们做的是"基于信任的执行"，不是传统Agent。
    
    传统Agent：能力授权 → 用户敢不敢是个问题
    我们：信任授权 → 关系越深，越敢放手
    
    而且执行只是我们产品进化的一个阶段，
    核心始终是"关系"，不是"功能"。
```

### Q: 技术壁垒是什么？

```
答：核心是"如何在记忆中建立关系"。
    
    这需要一套独特的架构设计，
    让AI能够：
    - 记住每个用户的独特之处
    - 理解情绪变化
    - 建立持续深化的关系
    
    具体实现是我们的核心竞争力。
```

> **而是在做一个更像朋友的AI。"**
>
> **它记得你、懂你、想你——**
> **而且会从认识你，进化到帮你做事。**
> **用户用得越久，越离不开。**
