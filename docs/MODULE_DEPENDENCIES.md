## 数据模型

### 记忆节点 (MemoryNode)

```typescript
interface MemoryNode {
  // 基础信息
  id: string;                    // 唯一标识
  content: string;               // 原始内容
  type: MemoryType;              // 类型: episodic | semantic | procedural | emotional | insight | identity
  category: MemoryCategory;      // 分类: work | life | skill | knowledge | emotion | ...
  
  // 向量表示
  embedding?: number[];          // 语义向量
  embeddingDimension?: number;   // 向量维度
  
  // 时间信息 (艾宾浩斯)
  createdAt: number;             // 创建时间
  lastRecalledAt?: number;       // 最后回忆时间
  recallCount?: number;          // 回忆次数
  strength?: number;             // 记忆强度 (0-1)
  retentionRate?: number;        // 保留率 (0-1)
  nextReviewAt?: number;         // 下次复习时间
  
  // 情感标记
  emotionalMarker?: {
    valence: number;             // 正负性 (-1 到 1)
    arousal: number;             // 激动程度 (0 到 1)
    dominance: number;           // 控制感 (0 到 1)
    labels?: string[];           // 情感标签
  };
  
  // 触发器
  triggers: Trigger[];           // 关联触发器
  
  // 关联
  associations: MemoryAssociation[];
  
  // 结晶化
  crystallizationLevel?: number; // 结晶等级 (0-5)
  isCoreMemory?: boolean;        // 是否核心记忆
  
  // 元数据
  importance?: number;           // 重要性 (0-1)
  tags?: string[];               // 标签
  metadata?: Record<string, unknown>;
}
```

### 触发器 (Trigger)

```typescript
interface Trigger {
  id: string;                    // 触发器ID
  memoryId: string;              // 所属记忆ID
  type: TriggerType;             // 类型: keyword | concept | emotion | time | context
  pattern: string | string[] | RegExp;  // 触发模式
  strength: number;              // 触发强度 (0-1)
  lastTriggered?: number;        // 最后触发时间
  triggerCount: number;          // 触发次数
  enabled: boolean;              // 是否启用
}
```

### 关联 (Association)

```typescript
interface MemoryAssociation {
  targetId: string;              // 目标记忆ID
  type: AssociationType;         // 类型: semantic | temporal | causal | emotional | trigger
  weight: number;                // 连接权重 (0-1)
  formedAt: number;              // 形成时间
  coActivationCount: number;     // 共同激活次数 (Hebbian)
  lastCoActivated?: number;      // 最后共同激活时间
}
```
