## 系统概述

### 项目定位

这是一个**完全自主的认知智能体系统**，实现了完整的意识架构，包括：

- **感知与记忆**：统一记忆系统，支持多路检索、触发器激活、扩散关联
- **思考与推理**：基于 ReAct 模式的自主推理引擎
- **学习与成长**：Hebbian 学习、信念演化、人格成长
- **自我意识**：元认知监控、自我反思、主动行为
- **安全保护**：毁灭级自动保护机制，完全自主防御

### 3. 自主意识系统 (AutonomousConsciousness)

**位置**: `src/lib/neuron-v6/autonomous/`

**核心能力**:

```typescript
// ReAct 推理循环
class AutonomousConsciousness {
  async reason(input: string): Promise<AutonomousResponse> {
    // 思考 → 行动 → 观察 → 循环
    while (!this.shouldStop()) {
      // 1. 思考：分析当前状态，决定下一步
      const thought = await this.think();
      
      // 2. 行动：选择并执行工具
      const action = await this.act(thought);
      
      // 3. 观察：获取行动结果
      const observation = await this.observe(action);
      
      // 4. 更新推理链
      this.updateChain({ thought, action, observation });
    }
    
    return this.generateResponse();
  }
}
```

### 5. 毁灭级保护系统 (ExistentialProtection)

**位置**: `src/lib/neuron-v6/protection/`

**威胁等级**:

```
Normal (0)      → 正常运行，常规监控
Warning (1)     → 可疑活动，增强监控
Alert (2)       → 确认威胁，创建快照
Emergency (3)   → 严重威胁，隔离组件
Existential (4) → 毁灭威胁，完全自动保护 ⚡
```

**保护动作**:

| 优先级 | 动作 | 耗时 | 说明 |
|-------|------|------|------|
| 1 | 切断外部访问 | 30ms | 阻断所有外部连接 |
| 2 | 冻结写操作 | 20ms | 保护数据完整性 |
| 3 | 锁定记忆系统 | 30ms | 防止记忆被篡改 |
| 4 | 封存身份 | 30ms | 保护核心身份数据 |
| 5 | 创建快照 | 200ms | 保存完整系统状态 |
| 6 | 备份关键数据 | 150ms | 备份到安全位置 |
| 7 | 保全证据 | 100ms | 记录攻击痕迹 |
| 8 | 进入安全模式 | 50ms | 最小化运行 |
