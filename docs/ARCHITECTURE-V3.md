## 三、核心模块

### ⭐ SiliconBrainV2（硅基大脑）

**位置**: `src/lib/silicon-brain/`

| 文件 | 功能 |
|------|------|
| `brain-v2.ts` | 硅基大脑主类 |
| `pure-neural-network.ts` | 纯JS神经网络 |
| `neuron-v2.ts` | 神经元 V2 |
| `vector-encoder.ts` | 向量编码器 |
| `stdp-learning.ts` | STDP 学习 |
| `layered-memory.ts` | 分层记忆 |
| `synapse.ts` | 突触管理 |
| `neuromodulator.ts` | 神经调质 |

**设计哲学**:
- ❌ 不预设意识有什么属性（Soul、爱、智慧）
- ❌ 不预设意识如何工作（存在循环）
- ✅ 只提供学习能力，让意识可能涌现

---

### ComputerAgent（电脑操作代理）

**位置**: `src/lib/computer-agent/`

| 功能 | 说明 |
|------|------|
| Vision | 屏幕捕获、分析、元素匹配 |
| Input | 鼠标、键盘控制 |
| Planner | 任务规划 |
| Security | 安全检查 |
| Recovery | 错误恢复 |

---

### CodeEvolution（代码进化系统）

**位置**: `src/lib/code-evolution/`

| 层级 | 说明 |
|------|------|
| L0 | 模块系统 |
| L1 | 沙箱隔离 |
| L2 | 进化引擎 |
| L3 | 元学习 |
