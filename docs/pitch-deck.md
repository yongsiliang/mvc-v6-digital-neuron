## 五、FAQ 预设

### Q1: 和ChatGPT有什么区别？

```
核心区别在于三个维度：

1. 记忆
   - ChatGPT: 会话级记忆，新对话从头开始
   - 我们: 永久记忆，越用越懂你

2. 主动性
   - ChatGPT: 被动等待指令
   - 我们: 主动思考，可能会先开口

3. 关系
   - ChatGPT: 对所有人一样
   - 我们: 和你建立独特的关系
```

### Q2: 如何保护用户隐私？

```
隐私保护措施：

1. 数据存储
   - 支持完全本地化部署
   - 用户可导出/删除所有数据

2. 向量化处理
   - 原始文本可加密存储
   - 只有向量参与计算

3. 透明度
   - 用户可查看所有记忆内容
   - 可手动编辑/删除记忆
```

### Q3: 商业模式是什么？

```
商业模式：

1. 个人版
   - 免费版：基础功能
   - Pro版：$9.9/月，更多主动性、更大记忆

2. 企业版
   - 团队神经元
   - 知识库集成
   - 按需定价

3. 开放平台
   - API调用
   - 插件生态
```

### Q4: 技术护城河是什么？

```
技术壁垒：

1. 架构创新
   - 双层空间架构（意识+记忆）
   - 链接强度动态激活机制

2. 专利潜力
   - 向量驱动的意识表示方法
   - 基于距离的记忆门机制

3. 数据积累
   - 用户越多，模型越准
   - 网络效应明显
```
