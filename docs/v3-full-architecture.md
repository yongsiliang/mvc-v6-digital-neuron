### 3.4 VSA Space (语义空间)

**文件**: `vsa-space.ts`

**职责**: 系统的"阳系统"（理性、符号、稳定）

```
┌─────────────────────────────────────────────────────────────┐
│                     VSA Semantic Space                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  维度: 10000维高维向量                                       │
│                                                             │
│  核心操作:                                                   │
│  ┌──────────────────────────────────────────────────────┐  │
│  │ 绑定 (Bind):   A ⊗ B = "A与B的关系"                    │  │
│  │ 捆绑 (Bundle): A ⊕ B = "A和B的集合"                    │  │
│  │ 置换 (Permute): ρ(A) = 位置编码                        │  │
│  │ 解绑 (Unbind): (A ⊗ B) ⊗ A⁻¹ ≈ B                      │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                             │
│  推理能力:                                                   │
│  • 类比推理: A:B :: C:?                                      │
│  • 关系推理: 提取语义关系                                    │
│  • 相似度计算: 余弦相似度                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键特性**:
- **独立理解**: 不依赖LLM的语义计算
- **符号推理**: 可组合、可推理的数学对象
- **稳定知识**: 概念向量相对稳定

## 五、使用示例

### 5.1 基本使用

```typescript
import { getConsciousnessAGI, resetConsciousnessAGI } from '@/lib/neuron-v3';

// 初始化系统
resetConsciousnessAGI();  // 确保干净状态
const agi = getConsciousnessAGI({
  userId: 'user-123',
  enableYinSystem: true,
  enableYangSystem: true,
  enableSelfCore: true,
});

// 处理输入
const response = await agi.process('你好，我想学习意识是什么');

console.log('响应:', response.content);
console.log('自我关联度:', response.subjectiveMeaning.selfRelevance);
console.log('阴阳平衡:', response.systemState.balance.balance);
console.log('意识水平:', response.systemState.consciousnessLevel);
```

### 5.2 获取系统报告

```typescript
const report = agi.getSystemReport();
```
