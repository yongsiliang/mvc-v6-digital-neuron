# 认知智能体 - 精简版

## 快速开始

```cmd
# 1. 安装依赖 (约30秒)
pnpm install

# 2. 配置 API Key
copy .env.example .env.local
notepad .env.local

# 3. 启动
pnpm dev

# 4. 访问 http://localhost:5000
```

## 示例任务

- 访问 https://httpbin.org/get 并告诉我返回了什么
- 用 POST 请求调用 https://httpbin.org/post

## 注意

完整功能需要配置 LLM API Key (ARK_API_KEY)
